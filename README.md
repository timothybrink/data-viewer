# Data Viewer

A simple telemetry server/UI, built in JS.

Very primative at this point. Features to add:

 - Data labels/units/etc
 - Multiple charts
 - Data export

So still a work in progress.

## Installation

Clone this repo, npm install, and run `npm start`
